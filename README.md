# DataAnalysisScripts
Scripts for quickly analyzing csv files

# LT.awk - LessThan

USAGE

cat "CSVFILE" |./LT.awk fieldtoanalyze valuetocheck

for example 

cat XOM.csv|./LT.awk 7 100

will only grab order sizes that were less than 100 shares since field 7 is Quantity.
